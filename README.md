# machine-learning 实验总览

这个仓库按实验编号整理了六组机器学习课程实验，覆盖手写数字识别、医学图像分类、显著性检测、生成模型、情感二分类和情感五分类。目录中的训练集、测试集、报告、结果图和提交文件会一并保留，模型训练权重与预训练权重缓存不纳入仓库。

## 实验索引

| 实验目录 | 主题 | 说明 |
| --- | --- | --- |
| [1.0](./1.0/README.md) | MNIST 手写数字识别 | 包含 TensorFlow/Keras、PyTorch 以及 NumPy/CuPy + PyQt 三类实现。 |
| [2.0](./2.0/README.md) | 医学图像二分类 | 以 MedImage 数据集为基础，对比 EfficientNet 与自定义网络方案。 |
| [3.0](./3.0/README.md) | 显著性检测 | 以 ResNet18 编码器-解码器网络完成 saliency map 预测与评估。 |
| [4.0](./4.0/README.md) | 生成模型 | 包含 CIFAR-10 上的 GAN 与扩散模型训练、推理和可视化脚本。 |
| [10.0](./10.0/README.md) | 情感二分类 | 从传统机器学习到 GloVe/LSTM、RoBERTa、DeBERTa 的多轮实验。 |
| [11.0](./11.0/README.md) | 情感五分类 | 基于大规模预训练语言模型完成多分类情感识别。 |

## 目录特点

- `1.0` 到 `4.0` 更偏向图像方向课程实验，包含原始数据、训练脚本、可视化结果与辅助说明。
- `10.0` 和 `11.0` 聚焦文本情感分析，分别对应二分类与五分类任务。
- 各实验内部保留了阶段性版本文件夹，便于回看模型迭代过程。

## 使用说明

1. 先进入对应实验目录阅读该目录下的 `README.md`。
2. 按实验说明准备环境与数据路径。
3. 运行训练或推理脚本时，优先检查脚本中的数据路径、模型名称和输出目录。

## 仓库托管说明

- 本仓库保留训练集、测试集、报告、曲线图、样例输出和提交文件。
- 模型训练权重、最佳检查点和预训练模型下载缓存默认不提交。
- 超过 GitHub 普通文件限制的大数据文件通过 Git LFS 托管。
